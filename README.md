# roundpong
Em 18 de outubro de 1958, o físico William Higinbotham apresentou no Brookhaven National Laboratory (EUA) o jogo Tennis for Two: dois jogadores com manípulos controlavam o ângulo de uma “raquete invisível” para rebater uma luz-ponto sobre uma rede, tudo exibido em um osciloscópio conectado a um computador analógico. Aqui uma implementação que pode ser executada em qualquer navegador moderno, usando apenas html, css e javascript.

A versão inicial desse jogo foi criada com o seguinte prompt, usando o kiro.dev:

"Consider you have a perfect circle as the limit for the game. Inside this circle, moving alongside its interior border you have two small arches (10 degrees each) representing the paddles of the players. One player will use the x and x keys to move his paddle and the other will use the left and right arrows to move his paddle. A small, round ball will start moving, from the center of the circle, to a randon portion of it, in a straight line. The players shoud use their paddles to avoid the ball to escpa the circle. Use only html, css and javascript to produce this game. Have separate files for html, css and javascript."

Em seguida, sua interface foi traduzida para o português.

Coloque todos os arquivos (index.html, style.css e script.js) em uma única pasta e abra o index.html no navegador de sua preferência.

Uma demonstração desse jogo está em https://brodtec.com/roundpong
